curso, definiendo de manera sintética cada uno de las técnicas
referenciadas anteriormente y citando un ejemplo en el marco de
la ciberseguridad y ciberdefensa.

carpeta “TRABAJOS” de este repositorio. Dentro de una carpeta con
nombre “GRUPO_X” donde grupo X es por ejemplo GRUPO_1**
